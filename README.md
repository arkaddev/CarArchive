# Car Archive

## Contents
- Summary
- Technologies
- Tools

## Summary
The program helps in car management. 
It archives the work that has been done on a given car,
and tells you when the next replacement is due (e.g. oil, brakes)

## Technologies
- Java (version 11)
- Spring Framework (version 3.1.2)
